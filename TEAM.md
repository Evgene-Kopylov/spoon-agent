## Project Information

### Spoon Agent
**Autonomous AI Agent for Cryptocurrency Trading Analysis**

Spoon Agent is a modular AI system for cryptocurrency trading analysis that combines:
- Real-time market data from Binance API
- News analysis via Tavily search
- Technical analysis using AI models
- Investment recommendations based on multi-factor analysis

### Key Features
- **Modular Architecture**: Built with LangGraph for flexible workflow orchestration
- **Multi-Source Data**: Combines market data, news, and technical indicators
- **AI-Powered Analysis**: Uses OpenAI models for intelligent trading insights
- **Production Ready**: Docker containers and production deployment configurations
