## Project Overview

Spoon Agent is an autonomous AI system designed for cryptocurrency trading analysis. It combines multiple data sources and AI models to provide comprehensive trading insights.

### Key Features
- **Multi-Source Data Integration**: Market data from Binance, news from Tavily
- **AI-Powered Analysis**: Uses OpenAI models for technical and fundamental analysis
- **Modular Architecture**: Built with LangGraph for flexible workflow orchestration
- **Production Ready**: Docker containers and deployment configurations

### Technology Stack
- Python 3.12+
- LangGraph
- OpenAI
- Binance API
- Tavily Search
- Docker
